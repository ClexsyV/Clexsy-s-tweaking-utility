# Clexsys PC Tweaker

Clexsys PC Tweaker is a powerful Windows optimization utility designed to maximize system performance, reduce latency, and remove intrusive bloatware.

> **IMPORTANT:** This is a compiled project. The source code is private and **cannot be requested**.

---

## Activation Required
To ensure secure use, this program requires a unique activation key linked to your Hardware ID (HWID).
- **How to get a key:** Message **Clexsy** on Discord to request your personal activation key.
- **Note:** Without a valid key, the program features will remain locked.

---

## Features

### Gaming & Performance
- **Input Latency Fix:** Optimizes peripheral DataQueueSizes for faster response times.
- **Mouse Optimization:** Applies a 1:1 hex curve and disables acceleration for raw input.
- **Kernel DPC Optimization:** Distributes timers and optimizes thread priorities.
- **NVIDIA Tweaks:** Enables specific driver optimizations like RID61684 and DLSS Indicators.

### System & Privacy
- **Telemetry Block:** Full disabling of Windows, Office, and Developer data collection.
- **Bloatware Removal:** Uninstalls pre-installed junk apps (3DBuilder, CandyCrush, etc.).
- **Update Control:** Completely disable or manage automatic Windows Updates.
- **Security Toggling:** Easily manage Windows Defender, SmartScreen, and VBS.

### Network & Cleaning
- **TCP Gaming Opt:** Disables Nagle's Algorithm and optimizes packet priorities.
- **System Deep Clean:** Automated removal of Temp files, Logs, and Update caches.
- **Windows Activation:** Built-in tools for activating Windows Home/Pro via KMS.
